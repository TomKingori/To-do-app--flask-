# flask To Do App

## Setup and installation

Make sure you have the latest version of Python installed.

```sh
git clone <repo-url>
```

```sh
pip install flask
```

```sh
pip install Flask-SQLAlchemy
```

```sh
pip install flask-login
```

## Running the app

```sh
python main.py
```

## Viewing The App
Go to `http://127.0.0.1:5000`
